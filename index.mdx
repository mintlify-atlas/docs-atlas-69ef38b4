GrapesJS is a powerful, open-source **web builder framework** that lets you embed a fully functional drag-and-drop editor into any web application. It provides a rich set of APIs and a modular architecture so you can build custom page builders, email editors, newsletter creators, and site generators — all without imposing a rigid UI on your users.

## What you can build

GrapesJS is a framework, not a finished product. You define the interface; it provides the engine. Common use cases include:

- **Page builders** — Drag sections, columns, and content blocks onto a canvas
- **Email/newsletter editors** — Compose responsive HTML emails visually
- **Template editors** — Let users customize pre-built layouts without touching code
- **Site generators** — Combine GrapesJS with a storage backend for full website management

## Quick setup

<Steps>
  <Step title="Install GrapesJS">
    Add GrapesJS to your project via npm or a CDN script tag.
  </Step>
  <Step title="Initialize the editor">
    Call `grapesjs.init({ container: '#gjs' })` with your configuration object.
  </Step>
  <Step title="Add blocks and components">
    Define drag-and-drop blocks and custom component types for your use case.
  </Step>
  <Step title="Connect storage">
    Configure the Storage Manager to persist and load project data from your backend.
  </Step>
</Steps>
